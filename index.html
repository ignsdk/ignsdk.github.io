IGNSDK?

IGN SDK is a software development kit designed specifically for application developers in IGOS Nusantara.

Multiplatform

IGN SDK can be installed on many popular Linux-based operating systems like Debian, Ubuntu, Archlinux, Slackware, and openSUSE. Of course, it's available by default on IGOS Nusantara.

API

Beside great HTML5 APIs, IGN SDK also offers access to various native system capabilities. You can easily, for example, access a MySQL database with IGN SDK JavaScript API.
